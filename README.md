# 🏥 Analyse de données hospitalières SAE — Identification d'un hôpital pilote MedTech

## 📌 Description du projet

Dans le cadre de ma formation en Bachelor Sciences & Ingénierie des Systèmes Numériques Santé, j'ai réalisé en équipe une étude de marché pour identifier un **hôpital pilote** adapté à une solution MedTech d'**évaluation des risques d'AVC**.

L'objectif était de convaincre le CODIR d'une startup MedTech en justifiant le choix de l'établissement hospitalier le plus pertinent pour tester la solution.

---

## 🛠️ Technologies utilisées

- **Python** — traitement et analyse des données
- **CSV** — manipulation de fichiers de données hospitalières

---

## 📂 Données utilisées

- **SAE 2024** (Statistiques Annuelles des Établissements de santé) — base de données réelle du Ministère de la Santé
- **Données INSEE 2025** — population par sexe et groupe d'âges

---

## 🔍 Ce que fait le script

1. Lecture et parsing du fichier CSV des données hospitalières (neurochirurgie)
2. Extraction des identifiants d'établissements (RS) et des ETP salariés
3. Calcul du total d'ETP par établissement
4. Identification de l'hôpital avec les ressources les plus adaptées

---

## 📊 Résultat

Identification de l'hôpital pilote le plus pertinent pour déployer une solution d'évaluation des risques d'AVC, sur la base des ressources humaines médicales disponibles.
